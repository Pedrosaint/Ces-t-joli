# 🎯 Color Usage Strategy

## 1️⃣ Primary Colors (Foundation)

These colors should dominate the website and define the overall brand feel.

| Color | Purpose |
|--------|----------|
| **Black** | Headings, navigation bar text, footer background |
| **Cream** | Main background color for sections |
| **Brown** | Primary buttons, highlights, section accents |

**Brand Feel:** Premium, structured, warm, trustworthy.

---

## 2️⃣ Secondary Colors (Accent & Support)

These colors should be used sparingly for emphasis and supporting elements.

| Color | Purpose |
|--------|----------|
| **Grey** | Paragraph text, borders, subtle backgrounds |
| **Sky Blue** | Informational badges, icons, academic highlights |
| **Red** | Call-to-action buttons (e.g., Apply Now), urgent highlights |

⚠️ Red should be used minimally for attention and urgency.

---

# 🏗 Website Color Structure

## Background System
- Main page background → **Cream**
- Alternate sections → **White**
- Footer → **Black**
- Cards → **White**

---

## Typography System
- Headings → **Black**
- Body text → **Grey**
- Links → **Brown**
- Link hover → **Red** or **Sky Blue**

---

## Button System

### Primary Button
- Background → **Brown**
- Text → **White**
- Hover → **Black**

### Secondary Button
- Background → **Sky Blue**
- Text → **White**
- Hover → **Brown**

### Alert / Important Button
- Background → **Red**
- Text → **White**

---

# 🧠 Brand Psychology

- **Black** → Authority & Trust
- **Cream** → Warmth & Approachability
- **Brown** → Stability & Identity
- **Sky Blue** → Education & Calmness
- **Red** → Action & Urgency

---

# ⚠️ Design Guidelines

Avoid:
- Using red as a dominant background color
- Overusing sky blue
- Mixing all colors in one section
- Excessive colorful buttons

Stick to:
- Cream + White + Black as base
- Brown as primary brand identity
- Sky Blue & Red as controlled accents

---

# 🏫 Expected Visual Direction

The website should appear:

- Premium
- Structured
- Disciplined
- Trustworthy
- Academically strong

This branding supports a serious, modern, and excellence-driven institution.
